# vue-task-tracker using Options Api (Vue 3 + Vue Router + JSON Server)

## Project setup

```zsh
pnpm install
```

### Compiles and hot-reloads for development

```zsh
pnpm run serve
```

### Compiles and minifies for production

```zsh
pnpm run build
```

### Run json-server

```zsh
pnpm run backend
```

See [Configuration Reference](https://cli.vuejs.org/config/).
